Chapter 17

Solenoid Magnetic Field

Magnetic Field Inside a Solenoid

A solenoid is a long coil of wire wound in a helix. When current flows through it, it creates a nearly uniform magnetic field inside:

B = μ_0 · n · I

Where:

  • B = magnetic field strength (in tesla, T)
  • μ_0 = permeability of free space = 4 · π · 10-7 T*m/A (approximately 1.26 · 10-6 T*m/A)
  • n = number of turns per unit length = N / L (turns/meter)
  • I = current through the wire (in amperes)

Key Properties

PropertyDescription
Field insideUniform, parallel to axis: B = μ_0 · n · I
Field outsideApproximately zero (for an ideal, infinitely long solenoid)
DirectionUse the right-hand rule: curl fingers in direction of current, thumb points to north (B direction)
Dependence on NB depends on turns per unit length (n = N/L), not total turns alone

Comparison with a Bar Magnet

An ideal solenoid produces a field pattern very similar to a bar magnet:

  • One end acts as a magnetic north pole, the other as south
  • Field lines emerge from the north end, loop outside, and re-enter at the south end
  • Inside the solenoid, the field is uniform and runs from south to north

How to Increase the Solenoid Field

  • Increase the current I
  • Increase the number of turns per unit length n (wind coils more tightly)
  • Insert a ferromagnetic core (iron core): this multiplies B by the relative permeability μ_r of the material, giving B = μ_0 · μ_r · n · I

Worked Example

Problem: A solenoid is 0.5 m long, has 1000 turns, and carries a current of 2 A. Find the magnetic field inside.

Solution:

n = N / L = 1000 / 0.5 = 2000 turns/m

B = μ_0 · n · I = 4 · π · 10-7 · 2000 · 2

B = 4 · 3.14 · 10-7 · 4000

B = 12.56 · 10-7 · 4000

B = 50240 · 10-7

B = 5.024 · 10-3 T = 5.024 mT

This is roughly 100 times Earth's magnetic field (about 50 microtesla).

Exam Tips:
- B inside a solenoid is UNIFORM (same everywhere inside).
- B depends on n = N/L (turns per meter), not just N.
- Doubling the current doubles B. Doubling the length while keeping N the same halves n and halves B.
- An iron core dramatically increases B (by factor of μ_r, which can be 1000+ for soft iron).
- Outside an ideal solenoid, B = 0.
